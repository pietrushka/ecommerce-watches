### Getting started

1. Clone the repo
```sh
git clone https://github.com/pietrushka/ecommerce-watches-next.js.git
```
2. Install NPM packages
```sh
npm install
```
3. Enter CMS url in `.env`
```
CMS_URL=https://e-commerce-watches-strapi.herokuapp.com
```
4. Start app
```sh
npm start
```
